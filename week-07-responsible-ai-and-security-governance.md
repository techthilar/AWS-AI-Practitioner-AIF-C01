**Domain 4 (14%) and Domain 5 (14%).** Two lighter domains, paired into one week.

## Topics — Domain 4 (Responsible AI)

- Features of responsible AI: bias, fairness, inclusivity, robustness, safety, veracity
- Tools: Amazon Bedrock Guardrails; SageMaker Clarify; SageMaker Model Monitor; Amazon A2I
- Responsible model selection (environmental/sustainability considerations)
- Legal risks of GenAI: IP infringement, biased outputs, loss of trust, hallucinations
- Dataset characteristics; bias vs. variance; overfitting vs. underfitting
- Transparent/explainable models; SageMaker Model Cards; human-centered design

## Topics — Domain 5 (Security, Compliance, Governance)

- Securing AI: IAM, encryption, Amazon Macie, AWS PrivateLink, shared responsibility model, Bedrock Guardrails
- Source citation & data lineage; data cataloging; Model Cards
- Secure data engineering: data quality, privacy-enhancing tech, access control, integrity
- Security/privacy: prompt injection, encryption in transit/at rest, data leakage prevention, output filtering, audit/logging, toxicity
- Hallucination detection & grounding: RAG grounding, output validation, confidence scoring
- Governance/compliance services: AWS Config, Inspector, Audit Manager, Artifact, CloudTrail, Trusted Advisor
- Data governance strategies; Generative AI Security Scoping Matrix
